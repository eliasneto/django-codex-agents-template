# Django Codex Agents Template

Template base para projetos Django com Docker, documentação técnica e agentes Codex.

## Estrutura

- `docs/` — documentação do projeto
- `.codex/agents/` — agentes personalizados
- `apps/` — apps Django
- `templates/` — templates do sistema
- `static/` — arquivos estáticos
- `docker/` — arquivos auxiliares de Docker
- `scripts/` — scripts de inicialização
